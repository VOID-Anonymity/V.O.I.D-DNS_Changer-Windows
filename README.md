# V.O.I.D™ DNS Changer (Windows Edition) 🛡️

**V.O.I.D™ DNS Changer** — это высокоэффективный скрипт на базе PowerShell для автоматической настройки и оптимизации DNS-серверов в операционных системах Windows. Скрипт создан для тех, кто ценит скорость отклика сети и хочет избавиться от ограничений, навязанных стандартными DNS-серверами провайдеров.

## 🚀 Возможности
* **Интеллектуальный подбор:** Тестирование задержки (ping) до крупнейших мировых узлов (Google, Cloudflare, Quad9) и специализированных игровых серверов (Xbox-DNS).
* **Автоматизация:** Больше не нужно блуждать по "Панели управления". Скрипт сам находит активный сетевой адаптер и вносит правки.
* **Игнорирование роутера:** Прямая настройка системы на работу с внешними DNS в обход локальных ограничений.
* **Чистый интерфейс:** Центрированный TUI (Text User Interface) в фирменном стиле V.O.I.D™.

## 🛠️ Системные требования
* **ОС:** Windows 10 / 11.
* **Права:** Запуск строго от имени **Администратора** (необходимо для изменения системных настроек сети).

## 📥 Инструкция по запуску

1. Скачайте файл `VOID_DNS.ps1`.
2. Нажмите на него **правой кнопкой мыши**.
3. Выберите пункт **"Выполнить с помощью PowerShell"** (Run with PowerShell).
4. Если система запросит подтверждение прав администратора — подтвердите.

*Примечание: Если скрипт мгновенно закрывается, убедитесь, что вы разрешили выполнение сценариев в вашей системе (Execution Policy).*
